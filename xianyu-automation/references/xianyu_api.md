# 闲鱼 URL 和接口参考

## 闲鱼搜索 URL

```
https://www.goofish.com/2fn?keyword=商品关键词&price=价格区间
```

## 闲鱼商品页 URL

```
https://www.goofish.com/item?id=商品ID
```

## 价格字段说明

- `price` - 商品价格
- `originalPrice` - 原价
- `freight` - 运费（买家承担）
- `discount` - 折扣比例

## 闲鱼常用术语

| 术语 | 含义 |
|------|------|
| 箱说全 | 包装箱、说明书齐全 |
| 九新 | 九成新 |
| 裸机 | 无配件包装 |
| 到手刀 | 买家收到货后要求降价 |
| 走平台 | 通过闲鱼平台交易 |
| 秒价 | 很快会卖出的价格 |
| 捡漏 | 以低于市场价买到 |
| 大刀 | 大幅降价 |
| 小刀 | 小幅降价 |
