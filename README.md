## 概述

闲鱼 AI 商贩助手，集成 Safari MCP 浏览器自动化能力，实现闲鱼商品监控、推荐、发布等核心功能。

## 技术栈

- **AI Agent**: OpenClaw (小宇 xiaoyu)
- **浏览器自动化**: Safari MCP (AppleScript + JavaScript)
- **编程语言**: Node.js, AppleScript
- **版本控制**: Git
